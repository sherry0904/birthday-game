# Memory Lane Platformer 專案設計文件 (Project Plan & Documentation)

> [!NOTE]
> 此文件作為本專案的核心知識庫（Source of Truth），詳細記錄了遊戲的架構、企劃與實作細節，方便未來開發或其他 Agent 接手延續開發。

## 1. 專案概述 (Project Overview)
本專案為一款為另一半量身打造的「生日驚喜」網頁跑酷遊戲。採用復古可愛像素風格（Pixel Art），並結合專屬回憶與客製化對話。
*   **技術棧**：Vanilla HTML5 Canvas, JavaScript (ES6 Modules), CSS3, Vite
*   **視覺風格**：16-bit 像素風、DotGothic16 點陣字體、視差滾動背景 (Parallax Scrolling)。

## 2. 遊戲流程與核心機制
1.  **創角前導 (Start Screen)**：
    *   顯示男孩（戴眼鏡、短髮）與女孩的正面像素圖供選擇。
    *   讓玩家輸入暱稱。
2.  **跑酷遊玩 (Platforming)**：
    *   橫向捲軸自動前進，使用空白鍵或點擊螢幕進行跳躍。
    *   躲避障礙物，收集專屬的回憶道具。每個關卡需收集 5 個道具以過關。
3.  **關卡過場 (Level Transition)**：
    *   顯示拍立得相框（未來可替換為真實照片）。
    *   顯示帶有玩家暱稱的專屬感性留言。
4.  **終點驚喜 (Final Screen)**：
    *   出現巨大的生日蛋糕。
    *   需要狂點擊螢幕（20次）來吹熄蠟燭。
    *   爆出彩帶並顯示最終的生日祝福與實體禮物提示。

## 4. 角色與美術設計 (Art & Sprites)
遊戲內建一個輕量級的像素渲染引擎（`drawSprite` 函數），透過字元矩陣與色碼表 (`PALETTE`) 即時渲染像素圖。

*   **男孩 (Boy)**：短髮、配戴眼鏡。包含正面待機 (`boyFront`)、跑動幀 (`boyRun1`, `boyRun2`)。
*   **女孩 (Girl)**：長髮/馬尾。包含正面待機 (`girlFront`)、跑動幀 (`girlRun1`, `girlRun2`)。

## 5. 資料夾結構與代碼架構
*   `index.html`：遊戲的 DOM 結構，包含 UI 重疊層 (`#ui-layer`)。
*   `style.css`：處理介面的 CSS 動畫、RWD 排版、字體引入與像素渲染 (`image-rendering: pixelated`)。
*   `main.js`：包含遊戲的 Game Loop (`requestAnimationFrame`)、實體類別 (`Player`, `Item`, `Obstacle`)、碰撞偵測、像素精靈資料 (`SPRITES`) 與輸入事件監聽。

## 7. 開發歷程與討論紀錄 (Development History)
本專案的誕生來自於一系列的討論與迭代：
*   **初期提案**：最初討論了幾種生日遊戲的點子，包含「記憶拼圖/解謎逃脫」、「專屬回憶跑酷 (Memory Lane Platformer)」以及「心理測驗/回憶問答」。最後選擇了「回憶跑酷遊戲」，因為它既有互動性，又能隨著關卡推進重溫交往過程。
*   **V1 開發 - 基礎機制建立**：
    *   確立了三個主要關卡的故事：Level 1 愛河煙火告白、Level 2 阿里山神木之旅、Level 3 溫馨同居日常。
    *   確立了最終的驚喜機制：狂點擊螢幕吹熄蛋糕蠟燭，並給予實體禮物的提示。
    *   初期先使用 Emoji 作為道具與角色的暫時替代品，建立遊戲的核心跑酷邏輯與 UI 流程。
*   **V1 迭代 - 轉換為純手工像素風**：
    *   決定捨棄 Emoji，改為純手工繪製的像素藝術 (Pixel Art) 來呈現「可愛像素風」。
    *   在 `main.js` 實作了輕量級的像素渲染引擎，利用字元陣列定義像素圖案並繪製於 Canvas 上。字體也一併更換為復古點陣字體 `DotGothic16`。
*   **V2 大升級 - 增強帶入感與視覺細節**：
    *   為了避免遊戲一開始過於突兀（像 Chrome 離線恐龍遊戲），決定加入「創角前導畫面」。
    *   實作了選擇「男孩（短髮戴眼鏡）」或「女孩」的功能，並允許輸入專屬暱稱，該暱稱會連動到過關台詞與最後的生日祝福中。
    *   全面升級了背景視覺效果，為每個關卡加入了具有「視差滾動 (Parallax Scrolling)」的立體像素風景（城市天際線、阿里山脈與森林、溫馨室內大窗戶）。
